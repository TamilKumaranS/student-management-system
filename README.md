# Student Management System

A Java-based console application developed to efficiently manage student records using Object-Oriented Programming principles. The system provides a simple menu-driven interface for performing various operations related to student information management.

## Features

* Add new student records
* View all student details
* Search for a student by ID or name
* Update existing student information
* Delete student records
* Menu-driven console interface
* Modular design using OOP concepts

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## Project Structure

The application is designed using separate classes to improve code readability and maintainability. It demonstrates the use of classes, objects, methods, and basic data handling techniques.

## How to Run

1. Clone the repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile and run the main Java file.
4. Follow the on-screen menu instructions to manage student records.
